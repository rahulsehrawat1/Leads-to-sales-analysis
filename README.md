# 2024 Sales & Lead Conversion Analytics

This project analyzes a full year of CRM data (January - December 2024) to track how leads move through the sales funnel and which regions or sources provide the highest ROI.

### üìà Key Insights
* **Funnel Efficiency:** Identified a specific drop-off point in the "Assessment" stage, suggesting a need for faster follow-ups.
* **Top Performers:** Found that Facebook-sourced leads had the highest volume, but Referral leads had a 40% higher conversion rate.
* **Global Trends:** Created a heatmap showing that while lead volume is high in South Asia, the fastest "Lead-to-Sale" time is in the Gulf region (UAE/Kuwait).
